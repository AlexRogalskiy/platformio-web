Get Started

Download and install the next generation PlatformIO IDE.

 Download PlatformIO IDE


PlatformIO IDE is built on top of GitHub's Atom "hackable" text editor. If you have Atom already installed please use PlatformIO IDE package for Atom.
Install PlatformIO tool with Command Line Interface (CLI).

 Install PlatformIO CLI


PlatformIO is written in Python and works on Mac OS X, Linux, Windows OS and ARM-based credit-card sized computers.
