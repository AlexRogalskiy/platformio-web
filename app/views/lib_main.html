Libraries A missing library manager for embedded platforms (Atmel AVR, MSP430 and ARM)

PlatformIO Library Manager allows you to organize external embedded libraries. You can search for new libraries via Command Line or this WebSite interfaces.
You don't need to bother for finding the latest version of library. Due to platformio lib update command you will have up-to-date external libraries.
Examples: Arduino MSP430 Arduino +i2c keyword:sensor author:"Adafruit Industries"
